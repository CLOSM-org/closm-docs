CLOSM i offers three pricing plans.

## Plan Comparison

| Feature | Free | Lite | Premium |
|---------|------|------|---------|
| **Monthly Price** | Free | $9.99 | $29.99 |
| **Create Artworks** | 10/month | Unlimited | Unlimited |
| **Save Artworks** | Up to 50 | Up to 500 | Unlimited |
| **Resolution** | Max 2K | Max 4K | Max 8K |
| **Merchandise** | - | Available | Available |
| **NFT Mints** | - | 3/month | 30/month |
| **Priority Rendering** | - | - | Available |
| **Ad-free** | - | Yes | Yes |

## Plan Details

### Free Plan

For those who want to try fractal art creation.

- Create up to 10 artworks per month
- Save up to 50 artworks
- All basic features available
- Publish to gallery

### Lite Plan

For those who want to seriously enjoy fractal art.

- Unlimited artwork creation
- 4K resolution output
- Merchandise creation
- 3 NFT mints per month
- Ad-free experience

### Premium Plan

For professional creators.

- All features unlimited
- 8K high resolution output
- 30 NFT mints per month
- Priority rendering for faster processing
- Dedicated support

## Changing Plans

### Upgrade

1. Open profile settings
2. Click "Change Plan"
3. Select new plan
4. Enter payment info

Upgrades apply immediately. Pro-rated difference is charged.

### Downgrade

1. Open profile settings
2. Click "Change Plan"
3. Select new plan

Downgrades apply at end of current billing period.

**Note**: If you exceed limits after downgrade:
- Artworks are not deleted
- New creation is restricted
- Some features limited until within limits

## Payment Methods

- Credit cards (Visa, Mastercard, American Express)
- Debit cards

## Cancellation Policy

- Cancel anytime
- Service available until end of billing period
- No refunds
