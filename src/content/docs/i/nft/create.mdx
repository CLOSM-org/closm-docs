This guide explains how to mint your fractal art as an NFT.

---

## Prerequisites

Before starting, confirm the following:

- [ ] Logged into CLOSM i
- [ ] Have at least one artwork
- [ ] MetaMask or WalletConnect-compatible wallet ready
- [ ] Gas fees (MATIC or ETH) available
- [ ] Subscribed to Lite or Premium plan

→ [Wallet setup details](/i/nft/setup/)

---

## Minting Steps

### Step 1: Open the NFT Page

Navigate to the "NFT" page from the sidebar or navigation menu.

### Step 2: Connect Your Wallet

Click the "Connect" button in the "Wallet Connection" section and approve in your wallet.

After connecting, you'll see:
- Your wallet address
- Current balance
- This month's usage (mint count by chain)

### Step 3: Select Artwork

Your artworks are listed in the "Artwork Selection" section.

- Select **one artwork** to mint as NFT
- Already minted artworks show a "Minted" badge and cannot be selected

:::note
You can only select one artwork at a time. To mint multiple artworks, mint them one by one.
:::

### Step 4: Choose Chain

Select the blockchain for minting.

| Chain | Features | Available Plans |
|:------|:---------|:----------------|
| Polygon | Low cost, fast, beginner-friendly | Lite / Premium |
| Base | Low cost, fast | Lite / Premium |
| Ethereum | High liquidity, widely adopted | Premium only |

:::tip
For beginners, **Polygon** or **Base** is recommended. Gas fees are low and minting completes in seconds.
:::

### Step 5: Check Gas Fees

The estimated gas fee for your selected chain is displayed.

Shown information:
- Estimated gas (MATIC/ETH)
- USD equivalent
- Current balance
- Network status (Normal/Congested/Very Congested)

:::caution
If your balance is less than the gas fee, the mint button will be disabled. Add more funds for gas.
:::

### Step 6: Execute Mint

Click the "Mint" button.

The following steps execute in sequence:

| Step | Description |
|:-----|:------------|
| **Preparing** | Generating metadata and preparing signature |
| **Awaiting signature** | Click "Confirm" in wallet popup |
| **Confirming** | Waiting for blockchain processing |
| **Completed** | Mint successful |

:::note[Processing Time Estimates]
- Polygon: A few seconds to 1 minute
- Base: A few seconds to 1 minute
- Ethereum: Several seconds to a few minutes
:::

## After Minting

### How to Verify

After successful minting, verify through:

1. **On CLOSM i**
   - "Minted" badge appears on the artwork
   - Transaction hash is viewable
   - Link to block explorer

2. **On OpenSea**
   - Access directly via "View on OpenSea" link on completion screen
   - Appears in your NFT collection

### NFT Metadata

The minted NFT contains:

| Field | Content |
|:------|:--------|
| Title | Artwork name |
| Description | Artwork description |
| Image | High-resolution image hosted on CDN |
| Region | Mandelbrot region classification (Main Cardioid, Seahorse Valley, etc.) |
| Zoom Level | Zoom level (Overview to Ultra Zoom) |
| Detail Level | Detail level (based on iterations value) |
| Color Palette | Color palette classification |
| Resolution | Image resolution |
| Chain | Minted chain |
| Creator | Creator information |

## If You Cannot Mint

### Checklist

| Symptom | Check |
|:--------|:------|
| Mint button disabled | Confirm wallet is connected |
| Cannot select artwork | Check if already minted |
| Cannot select chain | Check your plan (Free cannot use, Ethereum is Premium only) |
| Insufficient balance error | Add more gas funds |
| "Monthly limit reached" | Wait until reset on the 1st of next month |
| Transaction failed | Check network status and retry |
| Signature expired error | Reload page and retry |

→ [FAQ & Troubleshooting](/i/nft/faq/)
