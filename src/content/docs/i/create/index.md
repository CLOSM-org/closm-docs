This guide provides detailed instructions for creating fractal art on CLOSM i.

## Accessing the Creation Screen

1. After logging in, click "Create" in the header
2. The creation screen opens

You can also create derivative works:
- Select an artwork in the gallery
- Click "Create from this artwork"

## Screen Layout

### Main Canvas
The fractal renders in real-time.

### Parameter Panel (Right)
- **Position (x, y)**: Display coordinates
- **Zoom**: Magnification level
- **Iterations**: Calculation detail
- **Resolution**: Output image size

### Color Bar (Bottom)
Map colors using gradients.

## Basic Controls

### Navigation

| Action | Result |
|--------|--------|
| Drag | Move view position |
| Scroll | Zoom in/out |
| Click | Zoom centered on point |

### Adjusting Parameters

1. Enter values directly
2. Drag sliders
3. Fine-tune with arrow keys

## Saving Your Work

### Visibility

- **Public**: Visible in gallery to everyone
- **Private**: Only you can see it

### Metadata

- **Title**: Artwork name (required)
- **Description**: About the artwork (optional)
- **Tags**: Keywords for search (optional)

## Derivative Works

You can create derivatives from other users' works or your own. Derivative relationships are recorded and visible in the gallery.

## Tips

1. **Explore boundaries**: Complex, beautiful patterns exist at fractal boundaries
2. **Adjust iterations**: Higher zoom levels need more iterations
3. **Experiment with colors**: Same shape, different colors = different impression
