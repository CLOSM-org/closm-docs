## How to Check

View your order history at `/goods/orders`.

1. Open the Goods page (`/goods`)
2. Select **"Order History"** from the sidebar or navigation
3. View order list

:::tip[Direct Access]
You can also directly access `https://i.closm.llc/goods/orders`.
:::

## Order Details

Tap an order to view:

| Item | Description |
|:-----|:------------|
| **Order Number** | Needed for support |
| **Order Date** | When ordered |
| **Product Info** | Item, quantity, price |
| **Shipping Address** | Destination |
| **Tracking Number** | Shown after shipping (tap to track) |
| **Total Amount** | Amount paid |
