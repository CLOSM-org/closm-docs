## Orders

### Can I cancel after ordering?

You can cancel while status is "Processing" (before production). Request from Order History.

**Cannot cancel after production starts.**

### Can I change my order?

No changes after confirmation. Place a new order and cancel the existing one.

### Can I order multiple items at once?

Currently one item at a time.

## Products

### What material is the case?

High-quality polycarbonate (PC).

- Excellent impact resistance
- Lightweight
- Crisp print quality

### Will the print peel off?

UV printing provides durable finish for normal use.

May degrade with:
- Repeated strong friction
- Prolonged high temperatures
- Contact with organic solvents

### Is it wireless charging compatible?

Yes. Wireless charging works with case attached.

※ Speed may vary with some chargers.

## Troubleshooting

### "Create Merchandise" button not showing

| Cause | Solution |
|:------|:---------|
| Not logged in | Log in |
| No artworks created | Create an artwork first |

### Design editor not working

1. Reload page (F5 / Cmd+R)
2. Clear browser cache
3. Try different browser (Chrome / Firefox / Safari)
4. Disable extensions (ad blockers)

### Order won't complete

| Check | Solution |
|:------|:---------|
| Payment info correct? | Verify card details |
| Credit limit exceeded? | Contact card issuer |
| 3D Secure required? | Follow prompts |
| Address accurate? | Double-check |

### Can't track with tracking number

- **Just shipped**: May take 24-48 hours to update
- **Wrong number**: Check Order History
- **Use carrier site**: Direct link from email

### Package not arriving

1. **Check tracking** - Latest status
2. **Check for notices** - Mailbox/door
3. **Contact carrier** - With tracking number
4. **Contact support** - If unresolved

:::note[Contacting Support]
Always include your **order number** for faster help.
:::
