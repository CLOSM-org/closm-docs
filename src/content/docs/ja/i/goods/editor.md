デザインエディタでは、作品のスマホケースへの配置を自由に調整できます。

---

## 画面構成

```
┌─────────────────────────────────────────┐
│  ツールバー                              │
├─────────────────────┬───────────────────┤
│                     │                   │
│    プレビュー        │   コントロール     │
│    エリア           │   パネル          │
│                     │                   │
├─────────────────────┴───────────────────┤
│  確認ボタン                              │
└─────────────────────────────────────────┘
```

---

## 基本操作

| 操作 | 方法 |
|:-----|:-----|
| **移動** | 画像をドラッグ |
| **拡大/縮小** | ピンチ or スライダー |
| **回転** | 回転ハンドル or スライダー |
| **リセット** | リセットボタン |

---

## コントロールパネル

| コントロール | 機能 | 範囲 |
|:-------------|:-----|:-----|
| **X座標** | 水平位置の微調整 | -100% 〜 +100% |
| **Y座標** | 垂直位置の微調整 | -100% 〜 +100% |
| **拡大率** | サイズ変更 | 50% 〜 200% |
| **回転角度** | 角度指定 | 0° 〜 360° |

---

## デザインのコツ

### ✅ 推奨

| ポイント | 理由 |
|:---------|:-----|
| 余白を持たせる | ケース端まで詰め込みすぎない |
| 中心を意識 | 見せたい部分を中央に配置 |
| 3Dプレビュー確認 | 実際の見た目を確認 |

### ❌ 避けたい

| ポイント | 理由 |
|:---------|:-----|
| 重要な部分が端にある | 印刷時にカットされる可能性 |
| 極端な拡大 | 画質が粗くなる |
| カメラ穴を考慮しない | 機種によりカメラ位置が異なる |

:::caution[印刷範囲について]
プレビュー上の**点線内**が印刷保証範囲です。重要なデザイン要素は点線内に収めてください。
:::
