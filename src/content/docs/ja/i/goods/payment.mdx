## 料金の構成

```
最終価格 = 商品代金 + 配送料
```

| 項目 | 説明 |
|:-----|:-----|
| **商品代金** | 製品価格（製品選択時に表示） |
| **配送料** | 配送先・配送方法による |

:::note
チェックアウト時に合計金額が表示されます。
:::

---

## チェックアウトの流れ

カートから「チェックアウトへ進む」をクリックすると、チェックアウトページ（`/goods/checkout`）に進みます。

```
住所入力 → 送料計算 → 配送方法選択 → Stripe決済
```

---

### Step 1: 配送先住所を入力

| 項目 | 必須 | 説明 |
|:-----|:----:|:-----|
| **氏名** | ○ | 配送先の名前 |
| **国** | - | カートの製品国に自動設定（変更不可） |
| **郵便番号** | ○ | 日本: `123-4567` / アメリカ: `12345` |
| **都道府県/州** | ○ | ドロップダウンから選択 |
| **市区町村** | ○ | |
| **住所1** | ○ | 番地など |
| **住所2** | - | 建物名・部屋番号など |
| **電話番号** | - | |

:::note[配送国について]
配送国はカートに追加した製品の国に固定されます。異なる国への配送が必要な場合は、カートをクリアして製品を再選択してください。
:::

---

### Step 2: 送料を計算

住所を入力したら、**「送料を計算」** ボタンをクリックします。

- 郵便番号、都道府県/州 の入力が必須
- ボタンをクリックすると配送方法と送料が表示されます

---

### Step 3: 配送方法を選択

計算結果から配送方法を選択します。

| 表示内容 | 説明 |
|:---------|:-----|
| **配送方法名** | Standard など |
| **配送日数** | 例: 5-10 営業日 |
| **送料** | USD 表記 |

---

### Step 4: 支払い

配送方法を選択後、**「支払いへ進む」** をクリックすると **Stripe** の決済ページに遷移します。

| 方法 | 対応ブランド |
|:-----|:-------------|
| **クレジットカード** | Visa / Mastercard / American Express など |
| **デビットカード** | 上記ブランドのデビットカード |

:::caution[請求タイミング]
決済完了時に請求されます。**製造開始後のキャンセルはできません。**
:::

---

## 注文完了

決済が完了すると、注文完了ページ（`/goods/checkout/success`）が表示されます。

- 注文ID
- 注文内容
- 合計金額

注文状況は [注文履歴](/ja/i/goods/history/) で確認できます。
