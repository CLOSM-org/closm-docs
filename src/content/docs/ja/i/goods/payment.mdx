## 料金の構成

```
最終価格 = 商品代金 + 配送料
```

| 項目 | 説明 |
|:-----|:-----|
| **商品代金** | 製品価格（製品選択時に表示） |
| **配送料** | 配送先・配送方法による |

:::note
チェックアウト時に合計金額が表示されます。
:::

---

## チェックアウトの流れ

カートから「チェックアウトへ進む」をクリックすると、チェックアウトページ（`/goods/checkout`）に進みます。

### Step 1: 配送先住所を入力

| 項目 | 必須 | 説明 |
|:-----|:----:|:-----|
| **氏名** | ○ | 配送先の名前 |
| **郵便番号** | ○ | |
| **都道府県/州** | ○ | ドロップダウンから選択 |
| **市区町村** | ○ | |
| **住所1** | ○ | 番地など |
| **住所2** | - | 建物名・部屋番号など |
| **電話番号** | - | |

:::note
配送国はカートに追加した製品の国が自動的に設定されます。
:::

### Step 2: 配送方法を選択

配送先住所を入力すると、利用可能な配送方法と送料が表示されます。

- 配送方法により送料・配送日数が異なります
- ご希望の配送方法を選択してください

### Step 3: 支払い

配送方法を選択後、**Stripe** の決済ページに遷移します。

| 方法 | 対応ブランド |
|:-----|:-------------|
| **クレジットカード** | Visa / Mastercard / American Express など |
| **デビットカード** | 上記ブランドのデビットカード |

:::note[請求タイミング]
決済完了時に請求されます。製造開始後のキャンセルはできません。
:::

---

## 注文完了

決済が完了すると、注文完了ページ（`/goods/checkout/success`）が表示されます。

- 注文ID
- 注文内容
- 合計金額

注文状況は [注文履歴](/ja/i/goods/history/) で確認できます。
