## 料金の構成

```
最終価格 = 商品代金 + 印刷費用 + 配送料
```

| 項目 | 説明 |
|:-----|:-----|
| **商品代金** | ケース本体の価格 |
| **印刷費用** | フルカラー印刷の費用 |
| **配送料** | 配送先・配送方法による |

:::note
注文確定前に合計金額が表示されます。追加料金は一切かかりません。
:::

## 支払い方法

| 方法 | 対応ブランド |
|:-----|:-------------|
| **クレジットカード** | Visa / Mastercard / American Express / JCB |
| **デビットカード** | 上記ブランドのデビットカード |

:::note[請求タイミング]
注文確定時に**即時決済**されます。製造開始後のキャンセルはできません。
:::

## 配送先の登録

### 初回注文時

以下の情報を入力します:

1. **氏名**（漢字・ローマ字）
2. **郵便番号**
3. **住所**（都道府県・市区町村・番地・建物名）
4. **電話番号**

:::tip[住所の保存]
「この住所を保存する」にチェックを入れると、次回以降の入力が省略できます。
:::

### 2回目以降

保存済みの住所から選択、または新規入力ができます。

---

## 注文の流れ

1. **配送先住所** を入力または選択
2. **支払い方法** を選択
3. **注文内容** を最終確認
4. **「注文を確定」** をクリック

注文確定後、確認メールが送信されます。
