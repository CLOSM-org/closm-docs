CLOSM i は3つの料金プランを提供しています。

## プラン比較

| 機能 | Free | Lite | Premium |
|------|------|------|---------|
| **月額料金** | 無料 | $9.99 | $29.99 |
| **作品作成** | 月10作品 | 無制限 | 無制限 |
| **作品保存** | 50作品まで | 500作品まで | 無制限 |
| **解像度** | 最大 2K | 最大 4K | 最大 8K |
| **グッズ販売** | - | 可能 | 可能 |
| **NFTミント** | - | 月3回 | 月30回 |
| **優先レンダリング** | - | - | 可能 |
| **広告非表示** | - | 可能 | 可能 |

## 各プランの詳細

### Free プラン

フラクタルアートの作成を試したい方向けの無料プランです。

- 月に10作品まで作成可能
- 最大50作品を保存
- 基本的な機能をすべて利用可能
- ギャラリーへの公開可能

### Lite プラン

本格的にフラクタルアートを楽しみたい方向けのプランです。

- 作品作成が無制限
- 4K解像度での出力
- グッズ販売機能の利用
- 月3回のNFTミント
- 広告なしで快適に利用

### Premium プラン

プロフェッショナルなクリエイター向けのプランです。

- すべての機能が無制限
- 8K高解像度出力
- 月30回のNFTミント
- 優先レンダリングで待ち時間短縮
- 専用サポート

## プラン変更

### アップグレード

1. プロフィール設定を開く
2. 「プランを変更」をクリック
3. 新しいプランを選択
4. 支払い情報を入力

アップグレードは即時適用されます。日割り計算で差額のみ請求されます。

### ダウングレード

1. プロフィール設定を開く
2. 「プランを変更」をクリック
3. 新しいプランを選択

ダウングレードは現在の請求期間の終了時に適用されます。

**注意**: ダウングレード時に制限を超える作品がある場合：
- 作品は削除されません
- 新規作成が制限されます
- 制限内に収まるまで一部機能が制限されます

## 支払い方法

- クレジットカード（Visa, Mastercard, American Express）
- デビットカード

## キャンセルポリシー

- いつでもキャンセル可能
- 現在の請求期間の終了までサービスを利用可能
- 返金はありません

### Q: 学生割引はありますか？
