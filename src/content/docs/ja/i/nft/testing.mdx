CLOSM i には、**本番環境とは別のテスト環境**が用意されています。

テスト環境では、**実際のお金を使わずに** NFTミントの流れを体験できます。

---

## テスト環境とは

テスト環境は、本番リリース前の機能確認や操作練習のために用意された環境です。

### テスト環境URL

```
https://dev.i.closm.llc
```

### 本番環境との違い

| 項目 | テスト環境 | 本番環境 |
|:-----|:----------|:---------|
| URL | https://dev.i.closm.llc | https://i.closm.llc |
| 使用するお金 | **無料のテスト用トークン** | 実際の暗号資産 |
| NFTの価値 | なし（練習用） | 実際のNFT |
| ブロックチェーン | テストネット | メインネット |

:::tip[安心してお試しください]
テスト環境では実際のお金は一切かかりません。操作を間違えても問題ありません。
:::

## テスト環境の見分け方

テスト環境では、以下の表示で本番と区別できます。

### テストネットバナー

ページ上部に警告バナーが表示されます。

```
🧪 テストネット環境です
```

### 通貨表記の違い

| 本番環境 | テスト環境 |
|:---------|:----------|
| MATIC | tMATIC |
| ETH | tETH |

先頭に「t」が付いているのがテスト用トークンです。

### チェーン名の違い

| 本番環境 | テスト環境 |
|:---------|:----------|
| Polygon | 🧪 Polygon Amoy |
| Ethereum | 🧪 Sepolia |
| Base | 🧪 Base Sepolia |

## テストに必要なもの

### 1. テスト環境へのアクセス

テスト環境URL: **https://dev.i.closm.llc**

### 2. テスト用アカウント

テスト環境専用のアカウントでログインします。

- 本番環境のアカウントとは**別**です
- アカウント情報は担当者から提供されます

### 3. MetaMaskウォレット

NFTミントにはMetaMaskが必要です。

→ [ウォレット準備の詳細](/ja/i/nft/setup/)

### 4. テスト用トークン（無料）

テスト環境でのガス代には、**無料で取得できるテスト用トークン**を使います。

---

## テスト用トークンの取得方法

### Polygon Amoy（テスト用MATIC）

1. **[Polygon Faucet](https://faucet.polygon.technology/)** にアクセス
2. 「**Amoy**」を選択
3. MetaMaskのウォレットアドレスを貼り付け
4. 「**Submit**」をクリック

数分以内にテスト用MATICが届きます。

:::tip[取得量の目安]
0.5〜1 MATICあれば、複数回のテストに十分です。
:::

### Sepolia（テスト用ETH）

1. **[Sepolia Faucet](https://sepoliafaucet.com/)** にアクセス
2. 無料アカウントを作成してログイン
3. MetaMaskのウォレットアドレスを貼り付け
4. 「**Send Me ETH**」をクリック

:::note[1日の取得制限]
フォーセット（蛇口）には1日あたりの取得制限があります。足りない場合は翌日再度取得してください。
:::

### Base Sepolia（テスト用ETH）

1. **[Base Sepolia Faucet](https://www.alchemy.com/faucets/base-sepolia)** にアクセス
2. Alchemyアカウントでログイン
3. MetaMaskのウォレットアドレスを貼り付け
4. 「**Send Me ETH**」をクリック

または、Sepoliaで取得したETHを[Base Bridge](https://sepolia-bridge.base.org/)でブリッジすることもできます。

## MetaMaskにテストネットを追加

テスト環境を使うには、MetaMaskにテストネットワークを追加します。

### Polygon Amoy の追加手順

1. MetaMaskを開く
2. 画面上部のネットワーク名をクリック
3. 「**ネットワークを追加**」を選択
4. 「**ネットワークを手動で追加**」をクリック
5. 以下を入力して保存：

| 項目 | 入力値 |
|:-----|:-------|
| ネットワーク名 | `Polygon Amoy` |
| RPC URL | `https://rpc-amoy.polygon.technology/` |
| チェーンID | `80002` |
| 通貨記号 | `MATIC` |
| ブロックエクスプローラー | `https://amoy.polygonscan.com` |

### Sepolia の追加手順

| 項目 | 入力値 |
|:-----|:-------|
| ネットワーク名 | `Sepolia` |
| RPC URL | `https://rpc.sepolia.org` |
| チェーンID | `11155111` |
| 通貨記号 | `ETH` |
| ブロックエクスプローラー | `https://sepolia.etherscan.io` |

### Base Sepolia の追加手順

| 項目 | 入力値 |
|:-----|:-------|
| ネットワーク名 | `Base Sepolia` |
| RPC URL | `https://sepolia.base.org` |
| チェーンID | `84532` |
| 通貨記号 | `ETH` |
| ブロックエクスプローラー | `https://sepolia.basescan.org` |

## テストの手順

### Step 1: テスト環境にログイン

1. https://dev.i.closm.llc にアクセス
2. テスト用アカウントでログイン
3. 🧪マークが表示されていることを確認

### Step 2: ウォレットを接続

1. NFTページを開く
2. 「**ウォレット接続**」をクリック
3. MetaMaskで承認

### Step 3: 作品を選択

ミントしたい作品を1つ選択します。

### Step 4: チェーンを選択

| おすすめ | 説明 |
|:---------|:-----|
| **🧪 Polygon Amoy** | ガス代が安く、すぐに完了（初心者向け） |
| 🧪 Base Sepolia | Baseチェーンのテスト |
| 🧪 Sepolia | Ethereumのテスト |

### Step 5: ミントを実行

1. 「**ミント**」ボタンをクリック
2. MetaMaskで「**確認**」をクリック
3. 処理完了を待つ

### Step 6: 結果を確認

ミント完了後、以下で確認できます。

| 確認場所 | 方法 |
|:---------|:-----|
| CLOSM i | 作品に「Minted」バッジが表示 |
| テストネットエクスプローラー | トランザクションハッシュで検索 |

### テストネットエクスプローラー

| チェーン | エクスプローラーURL |
|:---------|:-------------------|
| Polygon Amoy | https://amoy.polygonscan.com |
| Sepolia | https://sepolia.etherscan.io |
| Base Sepolia | https://sepolia.basescan.org |

:::note[OpenSeaでの確認]
テストネットのNFTはOpenSea本番サイトには表示されません。[OpenSea Testnets](https://testnets.opensea.io) で確認できます。
:::

## よくある質問

### Q: テスト用トークンが届きません

フォーセットの1日制限に達している可能性があります：
- 翌日再度お試しください
- ウォレットアドレスが正しいか確認してください

### Q: ネットワークに接続できません

MetaMaskの設定を確認してください：
- ネットワーク情報が正しく入力されているか確認
- RPC URLが正しいか確認

### Q: トランザクションが失敗します

以下を確認してください：
- テスト用トークンの残高は十分ですか？
- 数分待ってから再度お試しください

### Q: テスト環境にアクセスできません

担当者にお問い合わせください：
- アクセス権限の確認が必要な場合があります
