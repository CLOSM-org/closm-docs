フラクタルアート作品をNFTとしてミントする手順を説明します。

---

## 事前準備

ミントを開始する前に、以下を確認してください：

- [ ] CLOSM i にログイン済み
- [ ] 自分の作品が1つ以上ある
- [ ] MetaMask または WalletConnect対応ウォレットを準備済み
- [ ] ガス代（MATIC または ETH）を用意済み
- [ ] Lite または Premium プランに加入済み

→ [ウォレット準備の詳細](/ja/i/nft/setup/)

---

## ミント手順

### Step 1: NFTページを開く

サイドバーまたはナビゲーションから「NFT」ページを開きます。

### Step 2: ウォレットを接続

「ウォレット接続」セクションで「接続」ボタンをクリックし、ウォレットで承認します。

接続後、以下が表示されます：
- ウォレットアドレス
- 現在の残高
- 今月の使用状況（チェーン別ミント回数）

### Step 3: 作品を選択

「作品選択」セクションに、あなたの作品が一覧表示されます。

- NFT化したい作品を**1つ選択**
- すでにNFT化された作品は「Minted」バッジが表示され、選択不可

:::note
一度に選択できる作品は1つです。複数の作品をNFT化する場合は、1つずつミントしてください。
:::

### Step 4: チェーンを選択

ミント先のブロックチェーンを選択します。

| チェーン | 特徴 | 利用可能プラン |
|:---------|:-----|:---------------|
| Polygon | 低コスト、高速、初心者向け | Lite / Premium |
| Base | 低コスト、高速 | Lite / Premium |
| Ethereum | 高流動性、広く普及 | Premium のみ |

:::tip
初めての方は **Polygon** または **Base** がおすすめです。ガス代が低く、数秒でミントが完了します。
:::

### Step 5: ガス代を確認

選択したチェーンのガス代見積もりが表示されます。

表示内容：
- 推定ガス代（MATIC/ETH）
- USD換算額
- 現在の残高
- ネットワーク状況（通常/混雑/非常に混雑）

:::caution
残高がガス代より少ない場合、ミントボタンが無効化されます。追加でガス代を用意してください。
:::

### Step 6: ミントを実行

「ミント」ボタンをクリックします。

以下のステップが順番に実行されます：

| ステップ | 説明 |
|:---------|:-----|
| **準備中** | メタデータの生成と署名の準備 |
| **署名待ち** | ウォレットのポップアップで「確認」をクリック |
| **確認中** | ブロックチェーンでの処理を待機 |
| **完了** | ミント成功 |

:::note[処理時間の目安]
- Polygon: 数秒〜1分程度
- Base: 数秒〜1分程度
- Ethereum: 数十秒〜数分程度
:::

## ミント完了後

### 確認方法

ミント成功後、以下で確認できます：

1. **CLOSM i 上**
   - 作品に「Minted」バッジが表示
   - トランザクションハッシュを確認可能
   - ブロックエクスプローラーへのリンク

2. **OpenSea**
   - 完了画面の「OpenSeaで確認」リンクから直接アクセス
   - あなたのNFTコレクションに表示

### NFTのメタデータ

ミントされたNFTには以下の情報が記録されます：

| 項目 | 内容 |
|:-----|:-----|
| 作品タイトル | 作品名 |
| 説明 | 作品の説明文 |
| 画像データ | CDNにホストされた高解像度画像 |
| Region | Mandelbrot領域の分類（Main Cardioid, Seahorse Valley等） |
| Zoom Level | ズームレベル（Overview〜Ultra Zoom） |
| Detail Level | ディテールレベル（iterations値に基づく） |
| Color Palette | カラーパレット分類 |
| Resolution | 画像解像度 |
| Chain | ミントしたチェーン |
| Creator | 作成者情報 |

## ミントできない場合

### チェックリスト

| 症状 | 確認事項 |
|:-----|:---------|
| ミントボタンが押せない | ウォレット接続済みか確認 |
| 作品が選択できない | すでにNFT化されていないか確認 |
| チェーンが選択できない | プランを確認（Freeは利用不可、EthereumはPremium限定） |
| 残高不足エラー | ガス代を追加で用意 |
| 「月間上限に達しました」 | 翌月1日のリセットを待つ |
| トランザクション失敗 | ネットワーク状況を確認し再試行 |
| 署名期限切れエラー | ページをリロードして再試行 |
