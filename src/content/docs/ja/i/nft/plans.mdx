NFTミントは、ご契約のプランによって利用可能なチェーンと月間ミント回数が異なります。

---

## プラン別機能

| 機能 | Free | Lite | Premium |
|:-----|:-----|:-----|:--------|
| NFTミント | - | ○ | ○ |
| Polygon | - | ○ | ○ |
| Base | - | ○ | ○ |
| Ethereum | - | - | ○ |
| 月間ミント（Polygon） | 0 | 3回 | 10回 |
| 月間ミント（Base） | 0 | 3回 | 10回 |
| 月間ミント（Ethereum） | 0 | 0 | 3回 |
| 作品保存数 | 10枚 | 無制限 | 無制限 |

:::note
Freeプランでは、NFT機能の説明ページを閲覧できますが、ミントはできません。
:::

## 月間ミント回数

### リセットタイミング

月間ミント回数は**毎月1日（UTC）** にリセットされます。

### 使用状況の確認

NFTページの「使用状況」セクションで確認できます：

- 今月のミント数（チェーン別）
- 残り回数
- リセット日時

### 回数の繰り越し

使用しなかった回数は**翌月に繰り越されません**。毎月リセットされます。

## 料金体系

### NFTミントにかかる費用

NFTミント自体に **CLOSM i への追加料金はかかりません**。

必要なのは：

| 項目 | 内容 |
|:-----|:-----|
| サブスクリプション | Lite または Premium プラン |
| ガス代 | ブロックチェーンの手数料 |

### ガス代の目安

| チェーン | 通貨 | 通常時 | 混雑時 |
|:---------|:-----|:-------|:-------|
| Polygon | MATIC | $0.01〜0.05 | $0.10程度 |
| Base | ETH | $0.01〜0.10 | $0.50程度 |
| Ethereum | ETH | $0.50〜2.00 | $5.00以上 |

:::tip
ガス代はネットワークの混雑状況で変動します。NFTページで実際の見積もりを確認してからミントしてください。
:::

## プラン別の推奨使い方

### Lite プラン

**月3回（Polygon）+ 月3回（Base）** のミント枠を活用：

- 厳選した作品をNFT化
- 低コストなチェーンでミント
- NFT機能のお試しに最適

### Premium プラン

**Polygon 10回 + Base 10回 + Ethereum 3回/月** を活用：

- 多くの作品をNFT化
- Ethereumで本格的なNFTコレクション構築
- 用途に応じてチェーンを使い分け

## プランのアップグレード

### Lite → Premium

プラン設定ページからアップグレードできます：

- 即座にEthereumミントが利用可能に
- Polygon/Baseの月間回数も増加（3回→10回）

### Free → Lite/Premium

Free プランからのアップグレードで、NFTミント機能が解放されます。

→ [プラン詳細・変更](/ja/i/account/)

## よくある質問

### Q: 月の途中でアップグレードした場合、回数はどうなりますか？

新しいプランの上限が即座に適用されます。すでに使用した分はカウントされたままです。

### Q: 回数上限に達したらどうなりますか？

翌月1日のリセットまでミントできません。追加購入のオプションはありません。

### Q: ダウングレードした場合はどうなりますか？

すでにミントしたNFTは影響を受けません。翌月以降、新プランの回数制限が適用されます。

### Q: PolygonとBaseで別々にカウントされますか？

はい。各チェーンごとに個別にカウントされます。Liteプランの場合、Polygon 3回 + Base 3回 = 合計6回のミントが可能です。
